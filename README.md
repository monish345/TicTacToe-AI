# Tic-Tac-Toe AI

A Python-based console Tic-Tac-Toe game featuring an unbeatable AI powered by the Minimax Algorithm.

## Features

- Player vs AI gameplay
- Unbeatable AI using Minimax Algorithm
- Console-based interface
- Win and draw detection
- Clean and beginner-friendly Python code

---

## Technologies Used

- Python 3
- Minimax Algorithm
- VS Code

## How to Run

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/TicTacToe-AI.git
```

### Open project folder

```bash
cd TicTacToe-AI
```

### Run the game

```bash
python main.py
```

---

## How the AI Works

The AI uses the Minimax Algorithm to evaluate all possible future moves and always select the optimal move.

This makes the AI impossible to beat if played correctly.
